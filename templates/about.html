About

Some Succulent details.

A test blog that I created for the Succulent gang to use as an internal blog. Like a bloge version of the whatsapp group.

The more traffic I can get on here the more I can test the platform and my ability to host publicly. As time goes on I will add more features and activate some of the current disabled features!

So post anything, something funny that happened today, a cool article that you've read, your progress on a personal project or any other musing.

Note that I have not added a photo selector for now but instead ask for a url to be provided and the photos are loaded on call from that url (e.g. https://encrypted-tbn0.gstatic.com/images?q=tbn:ANd9GcRBburtfM7ZDftVBzsTBO7sYwOCHlQ2VS1XuYLrTKV8pQ&s) To do this right click on the image and select 'copy image address'. So images (which are mandatory) must be in a place that is public (e.g. google photos public album) or just copy the url of an article image or a random relevant image from google images.

Ok get typing and posting and find those bugs!
